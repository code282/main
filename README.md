# Space Dodge 遊戲說明

## 功能概述

這是一個使用 HTML5 Canvas 開發的太空躲避遊戲（Space Dodge），玩家需要操控飛船躲避從上方落下的隕石。

## 主要功能

### 1. 遊戲核心機制
- **玩家控制**：使用鍵盤方向鍵（← →）或 A/D 鍵控制飛船左右移動
- **移動速度**：玩家移動速度為 8 像素/幀
- **碰撞檢測**：即時檢測玩家與隕石的碰撞，碰撞即遊戲結束

### 2. 隕石系統
- **隨機生成**：每隔一段時間（初始 800ms）自動生成隕石
- **隨機位置**：隕石從畫面上方隨機水平位置出現
- **速度遞增**：每得 10 分，隕石移動速度 +1，生成間隔縮短
- **視覺設計**：灰色圓形隕石，帶有白色斑點裝飾

### 3. 計分系統
- **得分機制**：每成功躲避一顆隕石得 1 分
- **分數顯示**：左上角即時顯示當前分數
- **最終分數**：遊戲結束時顯示最終得分

### 4. 遊戲狀態管理
- **遊戲結束**：碰撞後顯示「遊戲結束！按空白鍵重新開始」
- **重新開始**：按空白鍵重置遊戲（分數歸零、隕石清空、位置重置）
- **難度遞增**：隨著分數增加，遊戲難度動態提升

### 5. 視覺設計
- **玩家飛船**：藍色三角形太空船，使用 Canvas 路徑繪製
- **遊戲畫布**：800x600 像素，黑色背景，灰色邊框
- **顏色配置**：使用 COLORS 物件統一管理（黑、白、紅、灰、藍）

### 6. 技術實現
- **Canvas 2D 渲染**：使用 HTML5 Canvas API 進行圖形繪製
- **遊戲循環**：使用 `requestAnimationFrame` 實現流暢的 60fps 遊戲循環
- **事件監聽**：鍵盤按下/釋放事件控制玩家移動
- **碰撞檢測**：矩形碰撞檢測算法（AABB）

## 操作說明

| 按鍵 | 功能 |
|------|------|
| ← → 或 A/D | 左右移動飛船 |
| 空白鍵 | 遊戲結束後重新開始 |
